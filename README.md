# CODEALPHA_PHISING-AWARENESS
PHISING AWARENESS VIDEO
Phishing Awareness Video

Overview

This project is a phishing awareness video created to explain how phishing attacks work and how users can protect themselves from fake emails, suspicious links, and social engineering tricks.

The video was made frame by frame with simple visual scenes, basic animations, and smooth transitions.

Audio Note

At first, I tried using the voice generated through ChatGPT, but the audio output was not smooth enough for the final video and felt laggy/sloppy. So, I used Sarvam AI Text-to-Voice to generate a clearer and more natural voice-over for the final version.

Features

- Phishing awareness topic
- Frame-by-frame video creation
- Basic animations and transitions
- Clean cybersecurity-themed visuals
- Voice-over generated using Sarvam AI Text-to-Voice
- Suitable for LinkedIn, GitHub, and academic submission

Tools Used

- ChatGPT – content idea, video planning, and script support
- Sarvam AI Text-to-Voice – final voice-over generation
- Video editing workflow – scenes, animations, transitions, and audio
- GitHub – project documentation and sharing
